Detailed Description

Models digdug's harpoon weapon.

The harpoon has various states which need to be collected together so that its interactions with other objects in the game can be managed appropriately. The harpoon is created when the user presses the space key and deactivated when its deployment is complete.

Moves the harpoon tip by the desired step amount.

When digdug deploys the harpoon weapon, it is moved in the desired direction by incrementing/decrementing the x (or y) coordinate by the amount of step. The length variable is incremented when this function is called, storing the amount by which the cord has been extended.

void harpoon::unharpooning ( )

Stops the harpooning process.

This function is called to terminate the harpooning process when the harpoon reaches a wall, the cord length exceeds it's maximum allowed value (some constant) or the user releases the spacebar key.
