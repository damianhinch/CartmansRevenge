Detailed Description

Canvas is resonsible for the drawing of the characters and game screens.

Canvas draws the main character (digdug) and the monsters in their various stages of animation. The class draws the game screens, game layouts, text messages to the user and all other visual aspects of the game environment. It holds the screen layouts as its private data members.
